CSS.Features

What is it?

You say Progressive Enhancement, I say Graceful Degradation. No matter which side you are on, writing CSS which takes advantage of the wonderful new CSS features but still work well with many different browsers can be a pain. CSS.Features is a small Javascript library that makes this easy.

Usage

To use, just include the javascript file inside the <head> tag, like so: 

<script type="text/javascript" src="cssfeatures.js"></script>

CSS.Features adds a number of CSS classes to the <body> tag - one for each Feature. For example, if said feature were called NiftyFeature, then the HasNiftyFeature class would be added onto <body> if it's supported by your browser. A list of features that CSS.Features checks for is listed below.

Your Browser

Feature List

The features in green are the ones your browser supports. The ones in red are the ones it doesn't support.

BoxShadow

The BoxShadow feature lets you add a drop shadow to an element.

Usage:

-moz-box-shadow: 1px 2px 2px #222;
-webkit-box-shadow: 1px 2px 2px #222;
box-shadow: 1px 2px 2px #222;

The first value is the horizontal offset; second value is the vertical offset; the third is the size of the drop shadow; and the fourth is the color of the shadow.

PNGTransparency

The PNGTransparency feature is had by a browser if it renders PNG alpha transparency correctly.

Usage:

background: url(bg.png);

TextShadow

The TextShadow feature lets you apply drop shadow to text.

Usage:

-moz-text-shadow: 1px 2px 2px #222;
-webkit-text-shadow: 1px 2px 2px #222;
text-shadow: 1px 2px 2px #222;

The first value is the horizontal offset; second value is the vertical offset; the third is the size of the drop shadow; and the fourth is the color of the shadow.

BorderRadius

The BorderRadius features lets you apply rounded corners to an element.border-radius property adds rounded corners to elements.

Usage:

-webkit-border-radius: 5px;
-moz-border-radius: 5px;
border-radius: 5px;

To specify the radius for each specific corner, use: 

-webkit-border-top-left-radius 
-webkit-border-top-right-radius
-webkit-border-bottom-left-radius
-webkit-border-bottom-right-radius

for WebKit and

-moz-border-radius-topleft
-moz-border-radius-topright
-moz-border-radius-bottomleft
-moz-border-radius-bottomright

for Firefox.

BorderColors

The BorderColors feature lets you give a border multiple bands of colors, with which you can do things like beveled edges. Currently, this is only supported in the latest version of Firefox.

Usage:

border: 3px solid #000;
-moz-border-top-colors: #303030 #353535 #434343;
-moz-border-left-colors: #303030 #353535 #434343;
-moz-border-right-colors: #303030 #353535 #434343;
-moz-border-bottom-colors: #303030 #353535 #434343;
border-top-colors: #303030 #353535 #434343;
border-left-colors: #303030 #353535 #434343;
border-right-colors: #303030 #353535 #434343;
border-bottom-colors: #303030 #353535 #434343;

You must first specify the width of the border in a separate border or border-width declaration. Then, you need to specify all of the colors you want to use in succession for each side of the element. Each color specified takes up one pixel of the total thickness of the border.

BackgroundSize

The BackgroundSize feature lets you scale up or scale down a background image to the dimensions that you desire.

Usage:

-webkit-background-size: 100px 100px;
background-size: 100px 100px;

The first value is the width; the second is the height.

Opacity

The Opacity feature lets you change the alpha transparency of an element.

Usage:

opacity: 0.8;

The value is between 0 and 1 inclusive.

PositionFixed

The PositionFixed feature lets you fixed the position of an element in your viewport independent of the page scroll offset.

Usage: 

position: fixed;
bottom: 0px;
right: 0px;

The above example fixes the element to the bottom right corner of the browser's viewport.

RGBAColors

The RGBAColors features lets you colors in RGBA space - that is RGB plus alpha transparency.

Usage:

background-image: rgba(255, 255, 255, 0.4);
color: rgba(0, 0, 0, 0.8);
border-color: rgba(100, 100, 100, 1);

MinMaxHeightWidth

The MinMaxHeightWidth feature lets you specify maximum or minimum values for the height or width of an element.

Usage:

min-width: 500px;
max-width: 960px;
min-height: 460px;
max-height: 860px;

TextOverflow

You can use the text-overflow property to render the end of a string of text with an ellipsis ("...") right before it is clipped by the edge of its container.

Usage:

text-overflow: ellipsis;
-o-text-overflow: ellipsis;

Other possible values for text-overflow are ellipsis-word and clip.

WordWrap

The WordWrap features lets you control whether a long word can be broken onto the next line if it extends beyond its container's bounds.

Usage:

word-wrap: break-word;

Possible values are break-word and normal.

Transform

The transform feature lets you rotate, scale, translate, and even skew an element.

Usage:

-webkit-transform: rotate(5deg) scale(-1) translate(5px, -10px) skewX(5);
-moz-transform: rotate(5deg) scale(-1) translate(5px, -10px) skewX(5);

Transform takes any number of functions as transformation steps. The functions scale, scaleX, and scaleY take a number as argument. A negative number flips the element. The function rotate function takes a degree as argument, in the format seen in the example above. The functions translate, translateX, and translateY Take a length unit as argument. The functions skew, skewX, and skewY take a degree as argument.
